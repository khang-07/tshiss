# tshiss v0.0.1
> tshiss is a file converter application built in Tauri
### Setup
```
$ brew install ffmpeg
$ cd tshiss
$ yarn tauri build
```

### Uses
- Convert files
- Access absolute directories

### How to
1. Open tshiss
2. Choose input file
3. Choose output folder
4. Specify file extension
5. Press confirm
